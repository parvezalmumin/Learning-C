-) Breaking it down:

   #include <stdio.h>
                       This tells the compiler to include the Standard Input Output library.
                       Needed for functions like printf() and scanf().

   int main()
                       Every C program must have a main() function.
                       Program execution starts here.
                       int means the function returns an integer value.

  printf("Hello, World!\n");
                       Used to display output on the screen.
                      \n means new line.

return 0;
                    Tells the operating system that the program finished successfully.


#Escape Sequence 
- \n                 New line                        printf("Hello\nWorld"); → prints on two lines


#
Variables in C

  - A variable is a named memory location used to store data. 
  -Syntax: data_type variable_name = value;
   Example: int age = 20;
  -The value of a variable can change during program execution.


 # Arithmetic Operations in C

                   Used to perform mathematical calculations.

                                           Basic operators:

                                                           + (Addition)

                                                           - (Subtraction)

                                                           * (Multiplication)

                                                           / (Division)

                                                           % (Modulo – gives remainder)




 #  Float Data Type

             - Used to store decimal (fractional) numbers.

             - Example: float price = 12.5;

             - Takes 4 bytes of memory (on most systems).

             - Used when precision is needed in calculations.   


 #    Division in Float
                        - If both numbers are integers → result is integer division.

                        - If at least one number is float → result is float division.

                          -  Example:
                                      5 / 2 = 2 (integer division)
                                      5.0 / 2 = 2.5 (float division)            


# Arithmetic Operation in Float
                                 - Works the same as integers, but results include decimals.

                                         Example:
                                                   float a = 5.5, b = 2.0;
                                                   a / b = 2.75
                                                   a + b = 7.5



 #  Input in C

               Data is taken from the user using scanf().

               Syntax:    
               ____________________________________________
               |                                           |
               |         scanf("%d", &variable);           |
               |___________________________________________|

               Format specifiers:
                                 %d → int

                                 %f → float

                                 %c → char 
                                     


 #   Modulo Operator in C

                      Symbol: %

                      Gives remainder after division of two integers.

                      Example: 10 % 3 = 1

                      Works only with integers.     


 #    Hierarchy of Operators (Precedence Order)

                                        () → Parentheses

                                         *, /, % → Multiplication, Division, Modulo

                                         +, - → Addition, Subtraction

                                         When operators have the same precedence, left to right associativity is followed.
